# mapping_explorer
## INSTALLATION
```sh
$ pip install -r requirements.txt
```

```sh
rosrun capture_depth.py
python calib_extrinsic.py

rosrun check_depth.py
```

This is a ROS package. Compile it with catkin.
```sh
mkdir build
cd build
cmake ..
make

```
